# sns 프로젝트
### 1) 개요
SNS의 동작 흐름의 흥미가 가게되어 만들고 있는 프로젝트입니다.

<br>

### 2) 기간
2021년 3월 10일 - 현재 진행 중

<br>

### 3) 사용한 기술
#### Back
- java 11
- Spring Boot 2.6.4 및 Security
- Jpa, MariaDB
- Jwt

<br>

### 4) 프로젝트 기능
#### 4.1) 구현한 기능
- 회원 가입, 로그인(Jwt 토큰 방식)
- 회원 정보 조회
- 포스트 작성 및 수정, 삭제
- 댓글 작성 및 삭제
- 해시태그
- 좋아요 기능
 

#### 4.2) 추가 예정 기능
- 친구 기능

<br>
