# sns 프로젝트
### 1) 개요
SNS 백엔드 프로젝트 


#### 1.1) 만들게된 계기
본인은 SNS를 사용하지 않지만 많은 사용자와 데이터가 동 시간대에 다루어지는 부분과 현시대에서 많이 사용되는 의사소통의 수단이어서 SNS에 대해 관심을 가지게 되었습니다.
 <br>
그러다 보니 SNS에서 제공하는 기능들의 흐름에 대하여 흥미를 느끼게 되었고 이 프로젝트를 만들게 되었습니다. 

<br>

### 2) 기간
2022년 3월 10일 - 2022년 4월 12일

<br>

### 3) 사용한 기술
#### `Back-end`
- java 11
- Spring Boot 2.6.4
- Jpa, MariaDB
- Jwt

<br>

### 4) 프로젝트 기능
#### 4.1) 구현한 기능
- 회원 가입, 로그인(Jwt 토큰 방식)
- 회원 정보 조회
- 포스트 작성 및 수정, 삭제
- 댓글 작성 및 삭제
- 해시태그
- 좋아요 기능
- 팔로우 기능
